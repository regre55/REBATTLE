So, this is just an archive of Game Genie Codes, cheats for a specific emulator (bgb) and--because of unresolvable issues with both--patch files for Pokemon Yellow Legacy and Pokemon Crystal Legacy.
As for what they do--simple. They add the ability to rebattle regular trainers throughout both games. Yes, this is actually possible now.
Unfortunately, there's no way to turn off the effect if you use the patches. So, yeah even those have issues...but hey it works (more or less)

So, what you'll need is a copy of Pokemon Yellow Legacy/Pokemon Crystal Legacy, which are both QoL (Quality of Life) improvement hacks for Pokemon Yellow & Pokemon Crystal for the Game Boy & Game Boy Color respectively. 
You can find the downloads for the patch files for both in their Releases section:

Pokemon Yellow Legacy
https://github.com/cRz-Shadows/Pokemon_Yellow_Legacy

Pokemon Crystal Legacy
https://github.com/cRz-Shadows/Pokemon_Crystal_Legacy

Since these are just the patch files, you'll need to somehow *acquire* the original games (actually their roms) which you can find if you search for a bit. 
Sharing the roms is frowned upon as technically illegal (copyrights you know) so don't ask where to find the roms--they're not that hard to find OK?
Once you've gotten your rom, download the corresponding Legacy patch for that game. 
Then follow the instructions included with those patches. Voila! You now have a copy of the Legacy games!

Oh, but wait. You want to rebattle the regular trainers, don't you? No other reason to be here after all.

OK, so after acquiring your Legacy rom, now download the zip file here:
